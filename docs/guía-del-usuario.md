# 📖 Guía de Usuario - Project Pokémon

## Introducción

Project Pokémon es una aplicación web que permite crear equipos Pokémon, personalizarlos y utilizarlos para combatir contra otros jugadores mediante un sistema de matchmaking online.

Esta guía explica las funciones principales disponibles para los usuarios.

# 1. Registro e inicio de sesión

Para acceder a las funcionalidades de la aplicación es necesario disponer de una cuenta de usuario.

El usuario puede:

- Registrarse mediante correo electrónico y contraseña.
- Iniciar sesión con sus credenciales.
- Iniciar sesión utilizando una cuenta de Google.

# 2. Gestión del perfil

Desde la sección de perfil el usuario puede consultar y modificar su información personal.

Las opciones disponibles son:

- Cambiar avatar.
- Modificar nombre de usuario.
- Editar biografía.
- Seleccionar equipo favorito.
- Cerrar sesión.
- Borrar cuenta.

**Perfil de usuario**

# 3. Gestión de equipos

Cada usuario puede crear hasta cinco equipos Pokémon.

Para cada equipo es posible:

- Asignar un nombre.
- Modificar sus datos.
- Eliminar el equipo.

**Lista de equipos**

<img width="1918" height="930" alt="image" src="https://github.com/user-attachments/assets/3eef5bed-9f7a-4550-aef9-3d227f4cc686" />

# 4. Edición de Pokémon

Cada equipo puede configurarse con distintos Pokémon.

Las opciones de personalización incluyen:

- Selección de naturaleza.
- Selección de movimientos.
- Cambio de sexo.
- Activación de versión shiny.
- Asignación de apodo (nickname).

Los cambios se reflejan en tiempo real en la vista previa del Pokémon.

# 5. Búsqueda de combate

Una vez seleccionado un equipo, el usuario puede acceder al modo de combate online.

El sistema de matchmaking buscará automáticamente otro jugador disponible.

Cuando se encuentra un rival, ambos usuarios son emparejados y comienza el combate.

# 6. Sistema de combate

Los combates siguen las mecánicas modernas de Pokémon utilizando Pokémon de primera generación.

Durante cada turno el usuario puede:

- Utilizar uno de los movimientos disponibles.
- Cambiar de Pokémon.
- Rendirse.

La resolución de acciones se realiza de forma simultánea para ambos jugadores.

# 7. Chat durante el combate

Cada combate dispone de una sala de chat privada.

Los jugadores pueden comunicarse entre sí mientras el sistema muestra también los eventos y acciones que ocurren durante el combate.

# 8. Panel de administración

Los usuarios con rol de administrador disponen de un panel específico para la gestión de usuarios.

Desde esta sección es posible:

- Consultar todos los usuarios registrados.
- Buscar usuarios por nombre o correo electrónico.
- Modificar roles.
- Eliminar cuentas.

# Conclusión

Project Pokémon proporciona una experiencia completa de creación de equipos y combate online inspirada en las mecánicas competitivas modernas de Pokémon, permitiendo a los usuarios gestionar sus equipos, personalizar sus Pokémon y enfrentarse a otros jugadores en tiempo real.
